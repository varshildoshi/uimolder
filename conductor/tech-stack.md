# Technology Stack

## Core Frontend
- **Framework:** Angular v20 (Standalone Components)
- **Programming Language:** TypeScript
- **State Management:** Angular Signals
- **Utilities:** Angular CDK (Drag & Drop functionality)

## Styling & UI
- **Utility CSS:** Tailwind CSS v4
- **Component Library:** Angular Material
- **Icons:** Material Icons / Lucide (as per implementation)

## Backend & Infrastructure
- **Cloud Platform:** Firebase (Hosting, Auth, Database)
- **Library:** AngularFire (Firebase integration for Angular)
- **Server:** Express.js (For specialized AI backend capabilities)

## AI & Advanced Features (Planned)
- **AI Integration:** Google AI (Gemini) via Hashbrown AI
- **Features:** 
    - Natural Language Form/Layout Creation
    - Real-time AI response streaming
    - Automated UI generation from descriptions

## Architecture & Scalability (Planned)
- **Monorepo Management:** Nx (To support multi-app/library scalability and efficient builds)

## Quality Assurance
- **Test Runner:** Karma
- **Testing Framework:** Jasmine
